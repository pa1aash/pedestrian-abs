# Data Directory

This folder is gitignored. Download datasets manually.

## FZJ Pedestrian Dynamics Archive
Source: https://ped.fz-juelich.de/database

- `fzj/unidirectional/` — Unidirectional corridor flow (DOI: 10.34735/ped.2013.6)
- `fzj/bottleneck/` — Bottleneck flow (DOI: 10.34735/ped.2009.6)
- `fzj/bidirectional/` — Bidirectional corridor flow (DOI: 10.34735/ped.2013.5)

Format: txt files with columns: frame_id, ped_id, x, y

## ETH-UCY Trajectory Dataset
Source: Stanford Trajectron++ repo

- `eth-ucy/eth/`
- `eth-ucy/hotel/`
- `eth-ucy/univ/`
- `eth-ucy/zara1/`
- `eth-ucy/zara2/`

Format: txt files with columns: frame_id, ped_id, x, y
